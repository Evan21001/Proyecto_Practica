Teoría de Conjuntos

Conjunto

Se puede entender el concepto de conjunto como aquella colección de elementos, pertenecientes a la misma categoría, y cuya agrupación puede ser considerada o identificada en sí misma como un objeto.

Los objetos del conjunto se llaman elementos.

Ejemplos:

- La agrupación de las vocales agudas.

- Los nombres de una persona.

- Los síntomas del refuljo.

Notación de Conjuntos:

Los conjuntos se denotan con letras mayúsculas del alfabeto y se usan los corchetes para representar y definir conjuntos. En el interior de los corchetes se ubican los elementos que conforman el conjunto separado por comas.

Ejemplo:

A = { 2,4,6,8}

B = {A,G,O,S,T,O}

Determinación de Conjuntos:

Esto se refiere a la manera en que un conjunto puede ser expresado. Hay dos formas de determinar un conjunto:

- Por extensión

- Por determinación

- Determinación por extensión:

Para describir los elementos de un determinado conjunto se pueden mencionar uno a uno, a esto se le conoce como descripción por extensión. Se define el conjunto A conformado por las vocales, en este caso se puede describir el conjunto por extensión así:

A = {A,E,I,O,U}

Si un conjunto tiene demasiado elementos, se pueden usar los puntos suspensivos de la siguiente manera:

El conjunto A es el conjunto de los primeros 50 números enteros positivos.

A = {1,2,3,… ,48,49,50}

- Determinación por Comprensión:

En algunos casos los conjuntos pueden tener una variada cantidad de elementos y la descripción por extensión resultaría muy ardua. Se puede entonces describir los conjuntos mencionando las características que comparten los elementos que los conforman. Por ejemplo, si es el conjunto B conformado por todos los departamentos de Guatemala se puede escribir:

A = { x|x es un departamento de Guatemala}

En donde la barra | se lee como “tales que”. Así, la anterior expresión se lee: “B es el conjunto de los x, tales que x es un país”. En este caso el símbolo x es usado simplemente para representar los elementos del conjunto B.

Ejemplos:

C= { x|x es un número mayor que 5 y menor que 8}

D = { x|x es una letra del abecedario}

E = { x|x es una palabra de 5 sílabas}
